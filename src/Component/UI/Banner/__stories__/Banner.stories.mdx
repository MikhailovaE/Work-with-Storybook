## Position

You can use 3 positions of banners:

- right (default)

- left

- center

For change backgroundColor, use ```backgroundColor=...``` props.

### Controls:

<Canvas>
  <Story
    name="Controls"
    args={{
      title:"Lorem ipsum",
      description:"Suspendisse suscipit lacinia est ut volutpat.",
      backgroundColor:"#DEF3F4",
      position: 'right'
    }}
    argTypes={{
      position: {
        description: "Position of banner",
        control: "radio",
        options: ["right", "left", "center"],
        table: {
          type: { summary: "string" },
          defaultValue: { summary: "primary" },
        },
      },
    }}
  >
    {(args) => <Banner {...args} />}
  </Story>
</Canvas>
